                           REMAINING MANUAL STEPS
═══════════════════════════════════════════════════════════════════════════════

Before launching to production, complete these steps:

1. Environment Configuration
   ☐ Create backend/.env with production values
   ☐ Generate strong JWT secrets (64+ chars)
   ☐ Configure production database URL
   ☐ Set secure CORS origin

2. Database Setup
   ☐ Create production PostgreSQL database
   ☐ Run migrations: pnpm db:migrate:prod
   ☐ Optional: Seed demo data

3. Infrastructure Setup
   ☐ Obtain SSL certificates (Let's Encrypt)
   ☐ Configure domain DNS
   ☐ Setup firewall rules
   ☐ Configure Nginx with SSL

4. Deployment
   ☐ Build backend: pnpm build:backend
   ☐ Build frontend: pnpm build
   ☐ Start with PM2 or Docker
   ☐ Verify health endpoint

5. Post-Deployment
   ☐ Test all features
   ☐ Verify security headers
   ☐ Check logs
   ☐ Setup monitoring
   ☐ Configure backups
   ☐ Document access credentials

6. PM2 Configuration (if using PM2)
   ☐ Rename ecosystem.config.txt to ecosystem.config.js
   ☐ Review and adjust configuration
   ☐ Start: pm2 start ecosystem.config.js --env production

                            QUICK START COMMANDS
═══════════════════════════════════════════════════════════════════════════════

Development:
  pnpm install
  cd backend && cp .env.example .env && pnpm db:generate && pnpm db:migrate
  pnpm dev (Terminal 1)
  cd backend && pnpm dev (Terminal 2)

Production (Docker):
  docker-compose up -d
  docker-compose exec api pnpm db:migrate:prod
  docker-compose logs -f

Production (PM2):
  pnpm build:all
  mv ecosystem.config.txt ecosystem.config.js
  pm2 start ecosystem.config.js --env production
